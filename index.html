CODING

Coding is the process of writing instructions that computers can
understand. These instructions are called code or programs.
Code is used to create software, websites, games, and apps that
we use every day. It is also used to operate devices such as
laptops, desktops, phones, and tablets. Coding involves writing
instructions in a programming language that a computer can understand.
